## プロジェクト概要

Adlaire Framework Ecosystem（AFE）は、PHP/Go 言語ベースとした独自フレームワークの開発・提供を目的とするプロジェクトである。
本フレームワークは、Adlaire Group が関与するすべてのプロジェクト、および Adlaire Group 内部システムの共通基盤（ベースフレームワーク）として採用・活用される。

### 技術スタック（現フェーズ）

| 項目 | 内容 |
|------|------|
| **主要言語** | PHP 8.2 以上 |
| **将来言語** | Go（ハイブリッド移行フェーズにて採用予定） |
| **実行環境** | PHP-FPM + Nginx または PHP-FPM + Apache |
| **標準仕様** | **AFE-STD（AFE 独自標準仕様）** — 外部標準（PSR 等）への依存なし |
| **標準管理** | AFE-STD は独立プロジェクト [`/AFE-STD/`](../AFE-STD/) として管理 |

## 設計思想

| 設計思想 | 概要 | ステータス |
|---|---|---|
| **モジュラーコンポーネント駆動型モデルアーキテクチャ** | 各コンポーネントを独立・交換可能な単位として設計 | ✅ 確定 |
| **API インターフェース駆動型モデルアーキテクチャ** | すべての境界を API として定義。言語間連携の基盤 | ✅ 確定 |
| **DI コンテナ・サービスコンテナ駆動型モデルアーキテクチャ** | 依存関係を一元管理するコンテナを中核に据える | ✅ 確定 |
| **AFE-STD 全面採用** | AFE 独自標準仕様（AFE-STD）を唯一の標準体系として採用。PSR への準拠は行わない | ✅ 確定 |
| **長期安定性** | 後方互換性を重視。破壊的変更はメジャーバージョンアップのみ | ✅ 確定 |
